Getting Started

Best way to start with the API is to send a Post request to https://form-to-email-api.herokuapp.com/api/email with the following JSON object as body of the request

You can use API client like Postman to send the request. 

                                                                    {
                                                                        "name": "PavanS", //Senders's name 
                                                                        "receiver": "receiver1@example.com, receiver2@example.com", //Receiver's email address
                                                                        "subject": "Test mail", //Subject of the email
                                                                         "data": [{  // Data that should be displayed in email
                                                                          "field":"Age",
                                                                          "value":"21" //You can add as many as data objects 
                                                                        },{            // These are like form fields and its values
                                                                          "field":"Favourite food",
                                                                          "value":"Noodles"
                                                                        },]
                                                                      
                                                                      }

Now you can send POST JSON requests to https://form-to-email-api.herokuapp.com/api/email using any technology. But in here onwards I am going to demostrate how to send a request using jquery AJAX

How to Setup a Form

You can create a normal HTML form with unique ids for input fields

Send Ajax POST request

Make sure that you have included jquery.js file to the HTML file

I have created a javascript function called send() to trigger when submit button in clicked. Inside this function I made a mail object as I mentioned in getting started section and fillout respective data fields with the input fields of the HTML form.

Then I made an AJAX Json POST request to the server using "https://form-to-email-api.herokuapp.com/api/email" Url

Add onsubmit event listener to trigger the Send() function when submitting the Form

Responses for the request

If the request is successfull

                                    {
                                        "status": 1,
                                        "message": "Email Sent"
                                    }

You will receive a Json response like above if the request was successfull

If the request is unsuccessfull

                                    {
                                        "status": -1,
                                        "message": "Error Occured",
                                        "error": {
                                            "code": "EENVELOPE",
                                            "command": "API"
                                        }
                                    }

You will receive a Json response like above with the specific error if the request was unsuccessfull
